# Generate Parentheses (#22)

Given `n` pairs of parentheses, write a function to generate all combinations of well-formed parentheses.  

## Examples

### Example 1:

Input: `n = 1`  
Output: `["()"]`  
 
### Example 2:

Input: `n = 2`  
Output: `["(())", "()()" ]`  

### Example 3:

Input: `n = 3`  
Output: `["((()))","(()())","(())()","()(())","()()()"]`  

## Constraints:

`0 <= n <= 8`
